Reference Frames

If something moves at the same velocity as a charge relative to a reference point, that object sees the charge as stationary, while the reference point sees the charge as moving. But that creates two different magnetic field strengths, and two different magnetic forces. Which is correct?

Consider reference point A and reference point B observing a charge C. The velocity of frame B relative to A is vBA\vec{v}_\text{BA}vBA, the velocity of charge C relative to frame A is vCA\vec{v}_\text{CA}vCA, and the velocity of charge C relative to frame B is vCB\vec{v}_\text{CB}vCB. Assume both reference frames are inertial, so vBA\vec{v}_\text{BA}vBA is constant.
We have
vCA=vCB+vBA\vec{v}_\text{CA}=\vec{v}_\text{CB}+\vec{v}_\text{BA}vCA=vCB+vBA
the Galilean transformation of velocity (using vector addition)
Taking the time derivative, noting that vBA\vec{v}_\text{BA}vBA is constant, we have
dvCAdt=dvCBdt\frac{d\vec{v}_\text{CA}}{dt}=\frac{d\vec{v}_\text{CB}}{dt}dtdvCA=dtdvCB
meaning the acceleratiosn are the same. Thus, the force observed by the two reference frames should also be the same.

Now suppose reference frame A establishes a region of uniform magnetic field BA\vec{B}_\text{A}BA and no electric field. A positive charge qqq sent through it at velocity vCA\vec{v}_\text{CA}vCA experiences force FA=qvCABA\vec{F}_\text{A}=q\vec{v}_\text{CA}\times\vec{B}_AFA=qvCABA
But if reference frame B moves at the same vBA=vCA\vec{v}_\text{BA}=\vec{v}_\text{CA}vBA=vCA, then the relative velocity of the particle is 0, so there should be no magnetic force, but we know the charge experiences a force. So, there must be an electric field. Since we know FA=FB\vec{F}_\text{A}=\vec{F}_\text{B}FA=FB, we have
qvCABA=qEBEB=vBABAq\vec{v}_\text{CA}\times\vec{B}_A=q\vec{E}_B\implies \vec{E}_B=\vec{v}_\text{BA}\times\vec{B}_AqvCABA=qEBEB=vBABA
Thus, whether a field is observed as electric or magnetic depends on the motion of the reference frame relative to the sources of the field.

Faraday's Law Revisited

Recall a conducting loop moving into a magnetic field with velocity v\vec{v}v creates a magnetic force that indcues an emf =vLB\varepsilon=vLB=vLB. How does it appear to a reference frame in which the loop is stationary?
There must be an induced current, but where does it come from? We transform the fields into frame B
EB=EA+vBA=vBBB=BA1c2(vEA)=B\vec{E}_\text{B}=\vec{E}_\text{A}+\vec{v}\times\vec{B}_\text{A}=\vec{v}\times\vec{B}\\\vec{B}_\text{B}=\vec{B}_\text{A}-\frac{1}{c^2}\left(\vec{v}\times\vec{E}_\text{A}\right)=\vec{B}EB=EA+vBA=vBBB=BAc21(vEA)=B
There is no magnetic force, becuase the loop is stationary, but there is an electric force.
FE=qE=qvB=qvB,upward\vec{F}_E=q\vec{E}=q\vec{v}\times\vec{B}=qvB,\text{upward}FE=qE=qvB=qvB,upward
which agrees with frame A.
This field is the induced electric field. Faraday's law says that a changing magnetic field creates an electric field, but the magnetic field only changes in frame B, so the electric field is only observable in frame B.

Induced Magnetic Field

The induced magnetic field, like the induced electric field, is created from changes in electric flux.
Note that Faraday's Law says the induced magnetic field is in the negative direction but the Ampere-Maxwell equation shows the induced electric field is in the positive direciton.

Maxwell's Equations

The following four equations give a complete description of electric and magnetic fields:
Gauss's Law: charged particles create an electric field
EdA=Q0\oint\vec{E}\cdot d\vec{A}=\frac{Q}{\epsilon_0}EdA=0Q
Gauss' Law for magnetism: no isolated magnetic poles
BdA=0\oint\vec{B}\cdot d\vec{A}=0BdA=0
Faraday's Law: electric fields can be created by changing magnetic fields
Eds=dmdt\oint\vec{E}\cdot d\vec{s}=-\frac{d\Phi_\text{m}}{dt}Eds=dtdm
Ampere-Maxwell Law: currents and changing electric fields create magnetic fields
Bds=0Ithrough+00dedt\oint\vec{B}\cdot d\vec{s}=\mu_0I_\text{through}+\epsilon_0\mu_0\frac{d\Phi_\text{e}}{dt}Bds=0Ithrough+00dtde

An additional law, known as the Lorenz Force Law, tells how matter reacts to the electromagnetic fields.
F=q(E+vB)\vec{F}=q(\vec{E}+\vec{v}\times\vec{B})F=q(E+vB)

Electromagnetic Waves

Maxwell's theories predicted the existance of electromagnetic waves. We first have to assume there are fields with no sources, or in a source-free space.
The equations simplify to having no charges or currents:
EdA=BdA=0\oint\vec{E}\cdot d\vec{A}=\oint\vec{B}\cdot d\vec{A}=0EdA=BdA=0
Eds=dmdt\oint\vec{E}\cdot d\vec{s}=-\frac{d\Phi_\text{m}}{dt}Eds=dtdm
Bds=00dedt\oint\vec{B}\cdot d\vec{s}=\epsilon_0\mu_0\frac{d\Phi_\text{e}}{dt}Bds=00dtde

From the induced fields being perpendicular, we assume E\vec{E}E and B\vec{B}B are perpendicular, and are perpendicular to the direction of motion.
We also assume they travel as a plane wave, one that has the same field anywhere in a plane perpendicular to the direction of motion.

Suppose the direction of travel was the xxx-axis, E\vec{E}E was parallel to the yyy-axis, and B\vec{B}B to the zzz-axis. We check its consistency with the source-free Maxwell's equations
Cosider a box as a Gaussian surface, then the net flux is 0 because E\vec{E}E and B\vec{B}B enter and exit at the same rate.
If there was an xxx-component, then there would be changing field in the direction of motion, so the flux would not exactly cancel out.
