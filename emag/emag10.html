Reference Frames

If something moves at the same velocity as a charge relative to a reference point, that object sees the charge as stationary, while the reference point sees the charge as moving. But that creates two different magnetic field strengths, and two different magnetic forces. Which is correct?

Consider reference point A and reference point B observing a charge C. The velocity of frame B relative to A is v⃗BA\vec{v}_\text{BA}vBA​, the velocity of charge C relative to frame A is v⃗CA\vec{v}_\text{CA}vCA​, and the velocity of charge C relative to frame B is v⃗CB\vec{v}_\text{CB}vCB​. Assume both reference frames are inertial, so v⃗BA\vec{v}_\text{BA}vBA​ is constant.
We have
v⃗CA=v⃗CB+v⃗BA\vec{v}_\text{CA}=\vec{v}_\text{CB}+\vec{v}_\text{BA}vCA​=vCB​+vBA​
the Galilean transformation of velocity (using vector addition)
Taking the time derivative, noting that v⃗BA\vec{v}_\text{BA}vBA​ is constant, we have
dv⃗CAdt=dv⃗CBdt\frac{d\vec{v}_\text{CA}}{dt}=\frac{d\vec{v}_\text{CB}}{dt}dtdvCA​​=dtdvCB​​
meaning the acceleratiosn are the same. Thus, the force observed by the two reference frames should also be the same.

Now suppose reference frame A establishes a region of uniform magnetic field B⃗A\vec{B}_\text{A}BA​ and no electric field. A positive charge qqq sent through it at velocity v⃗CA\vec{v}_\text{CA}vCA​ experiences force F⃗A=qv⃗CA×B⃗A\vec{F}_\text{A}=q\vec{v}_\text{CA}\times\vec{B}_AFA​=qvCA​×BA​
But if reference frame B moves at the same v⃗BA=v⃗CA\vec{v}_\text{BA}=\vec{v}_\text{CA}vBA​=vCA​, then the relative velocity of the particle is 0, so there should be no magnetic force, but we know the charge experiences a force. So, there must be an electric field. Since we know F⃗A=F⃗B\vec{F}_\text{A}=\vec{F}_\text{B}FA​=FB​, we have
qv⃗CA×B⃗A=qE⃗B⟹E⃗B=v⃗BA×B⃗Aq\vec{v}_\text{CA}\times\vec{B}_A=q\vec{E}_B\implies \vec{E}_B=\vec{v}_\text{BA}\times\vec{B}_AqvCA​×BA​=qEB​⟹EB​=vBA​×BA​
Thus, whether a field is observed as electric or magnetic depends on the motion of the reference frame relative to the sources of the field.
The same logic can be applied if there is a E⃗A\vec{E}_\text{A}EA​ as well, with
E⃗B=E⃗A+v⃗BA×B⃗A\vec{E}_\text{B}=\vec{E}_\text{A}+\vec{v}_\text{BA}\times\vec{B}_\text{A}EB​=EA​+vBA​×BA​

If a charge qqq is stationary in reference A with B⃗A=0\vec{B}_A=0BA​=0, then we observe
E⃗A=14πϵ0qr2r^\vec{E}_\text{A}=\frac{1}{4\pi\epsilon_0}\frac{q}{r^2}\hat{r}EA​=4πϵ0​1​r2q​r^
From reference B, because B⃗A=0\vec{B}_\text{A}=0BA​=0, we have
E⃗B=E⃗A=14πϵ0qr2r^\vec{E}_\text{B}=\vec{E}_\text{A}=\frac{1}{4\pi\epsilon_0}\frac{q}{r^2}\hat{r}EB​=EA​=4πϵ0​1​r2q​r^
which shows Coulomb's law is still valid in a frame where the charge moves.
However, from reference B, the charge is moving at a velocity v⃗CB=−v⃗BA\vec{v}_\text{CB}=-\vec{v}_\text{BA}vCB​=−vBA​. By Biot-Savart law,
B⃗B=μ04πqr2v⃗CB×r^=−μ04πqr2v⃗BA×r^\vec{B}_B=\frac{\mu_0}{4\pi}\frac{q}{r^2}\vec{v}_\text{CB}\times\hat{r}=-\frac{\mu_0}{4\pi}\frac{q}{r^2}\vec{v}_\text{BA}\times\hat{r}BB​=4πμ0​​r2q​vCB​×r^=−4πμ0​​r2q​vBA​×r^
Factoring out the equation for electric field, we have
B⃗B=−ϵ0μ0v⃗BA×(14πϵ0qr2r^)=−ϵ0μ0v⃗BA×E⃗A\vec{B}_B=-\epsilon_0\mu_0\vec{v}_\text{BA}\times\left(\frac{1}{4\pi\epsilon_0}\frac{q}{r^2}\hat{r}\right)=-\epsilon_0\mu_0\vec{v}_\text{BA}\times\vec{E}_ABB​=−ϵ0​μ0​vBA​×(4πϵ0​1​r2q​r^)=−ϵ0​μ0​vBA​×EA​
Thus, the Biot-Savart law is simply a Coulomb electric field transformed into a moving reference frame. The general transformation equation is
B⃗B=B⃗A+(−ϵ0μ0v⃗BA×E⃗A)\vec{B}_\text{B}=\vec{B}_\text{A}+(-\epsilon_0\mu_0\vec{v}_\text{BA}\times\vec{E}_\text{A})BB​=BA​+(−ϵ0​μ0​vBA​×EA​)

Faraday's Law Revisited

Recall a conducting loop moving into a magnetic field with velocity v⃗\vec{v}v creates a magnetic force that indcues an emf ε=vLB\varepsilon=vLBε=vLB. How does it appear to a reference frame in which the loop is stationary?
There must be an induced current, but where does it come from? We transform the fields into frame B
E⃗B=E⃗A+v⃗×B⃗A=v⃗×B⃗B⃗B=B⃗A−1c2(v⃗×E⃗A)=B⃗\vec{E}_\text{B}=\vec{E}_\text{A}+\vec{v}\times\vec{B}_\text{A}=\vec{v}\times\vec{B}\\\vec{B}_\text{B}=\vec{B}_\text{A}-\frac{1}{c^2}\left(\vec{v}\times\vec{E}_\text{A}\right)=\vec{B}EB​=EA​+v×BA​=v×BBB​=BA​−c21​(v×EA​)=B
There is no magnetic force, becuase the loop is stationary, but there is an electric force.
F⃗E=qE⃗=qv⃗×B⃗=qvB,upward\vec{F}_E=q\vec{E}=q\vec{v}\times\vec{B}=qvB,\text{upward}FE​=qE=qv×B=qvB,upward
which agrees with frame A.
This field is the induced electric field. Faraday's law says that a changing magnetic field creates an electric field, but the magnetic field only changes in frame B, so the electric field is only observable in frame B.

We have Gauss's Law for the flux through a closed loop. This shows that electric field lines start and stop at charges.
(Φe)closed loop=∮E⃗⋅dA⃗=Qinϵ0(\Phi_\text{e})_\text{closed loop}=\oint\vec{E}\cdot d\vec{A}=\frac{Q_\text{in}}{\epsilon_0}(Φe​)closed loop​=∮E⋅dA=ϵ0​Qin​​

Induced Magnetic Field

The induced magnetic field, like the induced electric field, is created from changes in electric flux.
Note that Faraday's Law says the induced magnetic field is in the negative direction but the Ampere-Maxwell equation shows the induced electric field is in the positive direciton.

Maxwell's Equations

The following four equations give a complete description of electric and magnetic fields:
Gauss's Law: charged particles create an electric field
∮E⃗⋅dA⃗=Qϵ0\oint\vec{E}\cdot d\vec{A}=\frac{Q}{\epsilon_0}∮E⋅dA=ϵ0​Q​
Gauss' Law for magnetism: no isolated magnetic poles
∮B⃗⋅dA⃗=0\oint\vec{B}\cdot d\vec{A}=0∮B⋅dA=0
Faraday's Law: electric fields can be created by changing magnetic fields
∮E⃗⋅ds⃗=−dΦmdt\oint\vec{E}\cdot d\vec{s}=-\frac{d\Phi_\text{m}}{dt}∮E⋅ds=−dtdΦm​​
Ampere-Maxwell Law: currents and changing electric fields create magnetic fields
∮B⃗⋅ds⃗=μ0Ithrough+ϵ0μ0dΦedt\oint\vec{B}\cdot d\vec{s}=\mu_0I_\text{through}+\epsilon_0\mu_0\frac{d\Phi_\text{e}}{dt}∮B⋅ds=μ0​Ithrough​+ϵ0​μ0​dtdΦe​​

An additional law, known as the Lorenz Force Law, tells how matter reacts to the electromagnetic fields.
F⃗=q(E⃗+v⃗×B⃗)\vec{F}=q(\vec{E}+\vec{v}\times\vec{B})F=q(E+v×B)

Electromagnetic Waves

Maxwell's theories predicted the existance of electromagnetic waves. We first have to assume there are fields with no sources, or in a source-free space.
The equations simplify to having no charges or currents:
∮E⃗⋅dA⃗=∮B⃗⋅dA⃗=0\oint\vec{E}\cdot d\vec{A}=\oint\vec{B}\cdot d\vec{A}=0∮E⋅dA=∮B⋅dA=0
∮E⃗⋅ds⃗=−dΦmdt\oint\vec{E}\cdot d\vec{s}=-\frac{d\Phi_\text{m}}{dt}∮E⋅ds=−dtdΦm​​
∮B⃗⋅ds⃗=ϵ0μ0dΦedt\oint\vec{B}\cdot d\vec{s}=\epsilon_0\mu_0\frac{d\Phi_\text{e}}{dt}∮B⋅ds=ϵ0​μ0​dtdΦe​​

From the induced fields being perpendicular, we assume E⃗\vec{E}E and B⃗\vec{B}B are perpendicular, and are perpendicular to the direction of motion.
We also assume they travel as a plane wave, one that has the same field anywhere in a plane perpendicular to the direction of motion.

Suppose the direction of travel was the xxx-axis, E⃗\vec{E}E was parallel to the yyy-axis, and B⃗\vec{B}B to the zzz-axis. We check its consistency with the source-free Maxwell's equations
Cosider a box as a Gaussian surface, then the net flux is 0 because E⃗\vec{E}E and B⃗\vec{B}B enter and exit at the same rate.
If there was an xxx-component, then there would be changing field in the direction of motion, so the flux would not exactly cancel out.
